# ドキュメント管理ツール比較

## 除外済みツール

| ツール | 除外理由 |
|--------|----------|
| AsciiDoc | サイドバーが自動生成されず使いづらい |
| GitHub Wiki | 別リポジトリのため PR/レビュー不可 |

### GitHub Wiki の制約
- 本体リポジトリとは **別の Git リポジトリ** (`username/repo.wiki.git`)
- 本体への PR に Wiki の変更を含められない
- CI/CD での自動テストが難しい
- ブランチ機能なし（常に単一ブランチ）のため PR でレビューできない

### AsciiDoc の制約
- サイドバーが付与されないため使いづらい。不適格
